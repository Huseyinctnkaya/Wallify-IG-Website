Getting Started with Wallify IG

Learn how to seamlessly integrate your Instagram feed into your Shopify storefront and start driving social proof today.

info

Quick Setup

Most merchants have their first gallery live in under 5 minutes using our automated embedding tool.

Overview

Wallify IG is the modern standard for Instagram marketing on Shopify. We optimize image loading using lazy-loading techniques to ensure your site's Lighthouse scores remain high while displaying high-resolution content.

Manual Embedding (Liquid)

If you aren't using Shopify 2.0 Theme Blocks, you can manually embed your gallery by placing this snippet in your product.liquid or index.liquid file.

liquid 
<!-- Wallify IG Gallery Widget -->
{% render 'wallify-gallery', 
   gallery_id: 'main_homepage',
   layout: 'grid',
   columns: 4
%}

Styling with CSS Variables

Customize the appearance globally using our CSS custom properties. Add this to your main stylesheet:

css 
:root {
  --wallify-border-radius: 12px;
  --wallify-grid-gap: 16px;
  --wallify-accent-color: #ff5a3d;
}
