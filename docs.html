Getting Started

Learn how to integrate Wallify IG into your project. Wallify IG is a high-performance, developer-centric library designed to create immersive grid layouts for modern web applications.

Introduction

Wallify IG simplifies the process of creating responsive, masonry-style grids with integrated image optimization and lazy-loading features. Built with performance in mind, it leverages the power of GPU-accelerated transforms to ensure smooth scrolling even with thousands of items.

Quick Installation

You can install the library via your favorite package manager. Open your terminal and run the following command:

bash 
npm install @wallify/ig-core

Alternatively, if you prefer yarn: 

yarn add @wallify/ig-core

Basic Usage

Initialize the Wallify instance by passing a container element and your configuration options. Here's a simple example: 

import { Wallify } from '@wallify/ig-core';

// Initialize the grid
const wall = new Wallify({
  container: '#grid-container',
  columns: 3,
  gutter: 16,
  animate: true
});

wall.render();
info

Pro Tip

Ensure your container has a defined width. Wallify calculates item positions based on the parent's current bounding client rect.
